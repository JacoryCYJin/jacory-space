Hermes 这里主要指 Nous Research 的 Hermes Agent：一个可长期运行的个人 AI Agent。它不是单纯聊天工具，也不是单纯编程工具，重点是记忆、自动化、多平台入口、工具调用和多模型使用。
## 它是什么
Hermes Agent 由 Nous Research 开发。它可以装在本机、服务器或云环境里，通过 CLI、桌面端、Telegram、Discord、Slack、Email 等入口使用。
Nous Portal 是它的账号、订阅、API key、用量和模型入口；Hermes 4 是 Nous 的模型系列；Nous Chat 是聊天界面。
它更像“长期在线的个人 AI 助手/自动化执行器”，不是只针对代码项目的 Agent。

## 对比 OpenClaw
一句话区别：Hermes 是一个现成的长期 Agent；OpenClaw 是一个自托管 Agent 网关。

| 维度    | Hermes                       | OpenClaw                                     |
| ----- | ---------------------------- | -------------------------------------------- |
| 本质    | Nous Research 做好的 Agent 产品   | 把聊天软件、模型、工具和本机环境接起来的网关                       |
| 主角    | Hermes 这个 Agent 本身           | 你接进去的模型、工具和其他 Agent                          |
| 模型/付费 | 偏 Nous Portal 一站式：模型、额度、工具订阅 | 偏自己组装：自己接 OpenAI、Claude、OpenRouter 等         |
| 使用方式  | 装一个 Nous 生态里的 AI 助手          | 搭一个自己的 AI 助手中控                               |
| 适合场景  | 想直接用长期记忆、技能、工具调用             | 想从 Telegram/WhatsApp 等入口控制自己的电脑、服务器和各种 Agent |
